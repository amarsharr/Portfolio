bon appetit

Project Overview

I designed and developed Bon Appétit, a modern restaurant website using HTML and CSS. The website includes dedicated sections for the menu, about, and contact details, creating a smooth flow for visitors to explore the restaurant’s offerings.

A key focus of this project was responsiveness ensuring the layout adapts beautifully to different screen sizes, whether viewed on a desktop, tablet, or smartphone. With a clean and visually appealing design, Bon Appétit delivers an engaging browsing experience for food lovers.
